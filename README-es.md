# DtFkGenerator (Versão em português [aqui](README.md))
Generador de datos fake

## Acerca de
Este programa fue creado para auxiliar la generación de datos para testes de formularios y para carga de Base de Datos.
Com él es posible generar Guids (Uniqueidentifier), nombres (de ambos sexos, con apellidos o sin), fechas, CPFs e RUTs.

## Atajos de uso
Al ejecutar el programa, se pueden usar los siguientes atajos:

C - Copia al portapapeles

N - Generar nuevo valor

L - Generar lista de valores

F - Cerrar o programa (Esc también funciona)

1-6 - Seleccionar uma tab
