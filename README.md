# DtFkGenerator (Versión en español [aquí](README-es.md))
Gerador de dados fake

## Sobre
Este programa foi criado para auxiliar a geração de dados para testes de formulários e para carga de Banco de Dados.
Com ele é possível gerar Guids (Uniqueidentifier), nomes (de ambos os sexos, com sobrenomes ou sem), datas, CPFs e RUTs.

## Atalhos para uso
Ao executar o programa, podem ser usados os seguintes atalhos:

C - Copia para a área de transferência

N - Gerar novo valor

L - Gerar lista de valores

F - Fechar o programa (Esc também funciona)

1-6 - Selecionar uma aba
